# vendingmachine

## How to set up
  * after cloning, run "virtualenv env" and "source env/bin/activate"
  * "python -m pip install -r requirements.txt"
  * "python manage.py runserver"
  * go to localhost:8000
  * when making purchase make sure the coins inserted are in comma separated format, for example: 10,5,1
